## Current Focus / 当前重点

### [AstrBot ecosystem](https://github.com/AstrBotDevs/AstrBot)

This is where most of my recent attention goes: deployment, dashboard workflows, plugin ecosystems, multi-platform messaging integration, and the operational details that make bots usable in practice.

这是我最近投入最多精力的方向：部署、管理面板工作流、插件生态、多平台消息接入，以及让 bot 真正可用的运维细节。

### [astrbot-sdk integrations](https://docs.astrbot.app/)

I care about how integrations built around astrbot-sdk stay clear, maintainable, and developer-friendly instead of turning into fragile glue code.

我关注的是围绕 astrbot-sdk 的集成如何保持清晰、可维护、对开发者友好，而不是变成脆弱的临时胶水代码。

### Plugins, adapters, and bot-side workflows / 插件、适配器与 bot 工作流

Much of the real work lives in plugin architecture, platform adapters, and capability wiring, where long-term maintainability matters more than short-term hacks.

很多真正有价值的工作都发生在插件架构、平台适配器和能力编排这一层，这里长期可维护性比短期拼凑更重要。

### Documentation and operational clarity / 文档与运维清晰度

I also care about setup flows, docs, and operational guidance, because good tooling is only useful when other people can actually get it running.

我同样重视接入流程、文档与运维说明，因为再好的工具，如果别人跑不起来，也谈不上真正有用。

Public examples around this direction are still being curated on GitHub.
